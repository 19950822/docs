Description 
------------

Simulates keystroke events on the specified element, as though you typed the value key-by-key. 

Parameters 
-----------

<table><thead><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr></thead><tbody><tr><td>to</td><td>TestObject</td><td>Required</td><td>Represent a web element.</td></tr><tr><td><span>strKeys</span></td><td><span>String</span></td><td>Required</td><td><span>The combination of keys to type.</span></td></tr><tr><td><span>flowControl</span></td><td><span>FailureHandling</span></td><td>Optional</td><td><span>Spec</span><span>ify </span><a>failure handling</a><span> schema to determine whether the execution should be allowed to continue or stop.</span></td></tr></tbody></table>

Example 
--------

You want to select all text in "txt_Comment" element by pressing Ctrl+A.

import com.kms.katalon.core.mobile.keyword.MobileBuiltInKeywords as Mobile

import com.kms.katalon.core.testcase.TestCaseFactory as TestCaseFactory

WebUI.closeBrowser()
